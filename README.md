# Luyện Cộng Số

App web đơn giản giúp trẻ em luyện phép cộng số có 1 hoặc 2 chữ số.

## Cách chạy

Mở file `index.html` bằng trình duyệt, hoặc chạy:

```bash
python3 -m http.server 8080
```

Rồi truy cập: http://localhost:8080

## Tính năng

- Hiển thị bài toán dạng `4 + 7 = ?`
- 3 lựa chọn đáp án
- Phản hồi đúng/sai ngay lập tức
- Đếm số câu đúng và sai
